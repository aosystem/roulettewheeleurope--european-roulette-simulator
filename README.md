# European Roulette Wheel: Minimalist Simulator 🎡🇪🇺

A streamlined and practical **European-style Roulette Wheel** built with **Flutter**. This utility is designed for users who prefer the traditional "Single Zero" layout, providing a reliable, high-quality wheel simulation without the complexity of a betting table or chip management.

## 🌟 Key Features
- **Authentic European Layout:** Features the classic 37-pocket configuration (0 and numbers 1-36).
- **Minimalist Design:** Focuses purely on the wheel functionality for a fast, focused, and distraction-free experience.
- **Voice Announcements:** Integrated Text-to-Speech (TTS) to call out results, adding an immersive touch to every spin.
- **Customizable Delay:** Adjustable timing before the final result is revealed, allowing for a build-up of suspense and anticipation.
- **Global Support:** Fully localized in **50+ languages**, catering to international users and events.

## 💡 Best For...
- **Professional Setups:** Using a physical betting table while letting the app act as a fair, digital dealer.
- **Decision Making & Games:** Adding a high-quality visual element to board games or random selection tasks.
- **Probability Study:** Demonstrating the mechanics and odds of the European (Single Zero) system.

## 🛠 Technical Overview
- **Framework:** Flutter
- **Language:** Dart
- **Logic:** Precise randomization engine based on the 37-slot European wheel distribution and state-driven spin animations.

*Note: This repository contains the core logic and source code. Assets (images, sound effects, and localization files) are excluded to protect intellectual property.*
